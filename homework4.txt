2. Why do Github users need an SSH key pair? (Users can use either SSH key pair or github account info.) Please write down a brief explanation of SSH key.
SSH Key는 서버에 접속할 때 비밀번호 대신 key를 제출하는 방식이다. 이가 필요한 이유는 비밀번호를 이용하는 것보다 높은 수준의 보안을 유지할 수 있으며, 로그인 없이 자동으로 서버에 접속할 수 있기 때문이다. SSH key pair은 개인키와 공개키로 구성되며, 개인키는 사용자가 사용자의 컴퓨터에 보관할 때, 공개키는 서버인스턴스를 만들 때 등록해야한다.
