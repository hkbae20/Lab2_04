2. Why do Github users need an SSH key pair? (Users can use either SSH key pair or github account info.) Please write down a brief explanation of SSH key.

github에 뭔가를 하기 위해서 자신을 식별해야 할때 사용자 이름과 비밀번호를 매번 입력해야한다.
이 SSH키는 개인 컴퓨터에 키만 저장되어있으면 바로 권한이 부여되기 때문에 그 불편함을 줄일 수 있다.
다시말해서 SSH키는 github에 자신을 식별하고 권한을 얻을때 사용되는 키이며 이를 개인컴퓨터에 저장하여 편리하게 사용할 수 있다.
