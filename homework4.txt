1. What is .gitignore? You can write the answer either in Korean or English.
.gitignore은 프로젝트에서 관리할 필요가 없는 파일 목록을 지정하는 파일이다. 로그파일, 프로젝트 설정파일 등 관리할 필요가 없는 파일들을 git이 tracking하지 않도록 .gitignore을 설정한다.
2. Why do Github users need an SSH key pair? (Users can use either SSH key pair or github account info.) Please write down a brief explanation of SSH key.
SSH 서버에서 클라이언트를 인증하기 위해 패스워드를 사용할 수도 있지만 SSH key가 더 안전하고 검증된 인증방식이므로 SSH key를 사용한다. SSH key pair는 private key와 public key로 이루어져 있는데 public key는 서버에 업로드 되고 클라이언트를 인증할 때 private key를 통해 인증한다.
