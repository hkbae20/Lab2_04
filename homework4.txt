1. What is .gitignore? You can write the answer either in Korean or English.
.gitignore는 개인정보 등과 같이 사용자가 git에서 track하기 싫은 파일을 지정해주는 것이다(다른 사람과 공유하거나 기록하지 않기 위해).
.gitignore에서 git에서 track하기 싫은 파일의 경로나 파일의 종류 등을 다 적어놓을 수 있고, git은 이러한 파일들을 track하지 않고 무시하게 된다.
github에 gitignore 프로젝트로 들어가면 다른사람들이 만들어놓은 gitignore 파일을 다운받을 수도 있고, 또는 직접 gitignore 파일을 만들어서 사용해도 된다.

2. Why do Github users need an SSH key pair? (Users can use either SSH key pair or github account info.) Please write down a brief explanation of SSH key.
github에 접속하기 위해 매번 아이디와 비밀번호로 로그인해야하는 부담을 줄이기 위해 SSH key pair를 이용하게 된다.
SSH key는 Secured Shell key의 약자로 시스템간의 암호화된 인증(통신)을 위해 사용하는 SSH protocol에 사용되는 key이다.
