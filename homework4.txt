1. What is .gitignore? You can write the answer either in Korean or English.
.gitignore 란 깃에서 특정 혹은 디렉토리를 관리 대상에서 제외할 때 사용하는 파일이다.
이 파일 안에 적힌 내용들은 Git 에서 관리 하지 않겠다는 것을 의미한다.
2. Why do Github users need an SSH key pair? (Users can use ei+ther SSH key pair or github account info.) Please write down a brief explanation of SSH key.
Github를 사용 할 때 자신의 저장소에 파일을 업로드 해야하는 경우, 저장소를 수정할 수 있는 권한이 있다는 것을 인증하는 과정이 필요하다.
이러한 사용자 인증 과정에 SSH key를 이용한다. 이를 이용하면 서버에 접속할 때 비밀번호 대신 key를 제출하게 되어, 같은 컴퓨터에서 작업을 하면 로그인을 그때마다 안 해도 된다.
SSH key 는 public key와 private key로 이루어진다. 키를 생성하면 앞의 두 종류의 키가 만들어 진다. 이 중 private key는 사용자 컴퓨터에, public key는 git 서버가 설치된 컴퓨터에 존재하여
인증을 시도 할 때 두 키가 일치하는 지를 확인한다.
