2. Why do Github users need an SSH key pair? (Users can use either SSH key pair or github account info.) Please write down a brief explanation of SSH key.

SSH key는 클라이언트를 인증해줄 수 있는 수단으로, 공개키와 개인키 한 쌍으로 이루어진다. 공개키는 Git server에 등록되는 공개적인 키로, 이는 개인키로 암호화된 내용을 복호화하거나 특정 정보를 직접 암호화하여 개인키로 복호화할 수 있는 데이터를 생성하는 데에 활용된다. 이처럼 공개키-개인키 방식은 특정 정보를 암호화할 경우 개인키에 이상이 있으면 반드시 오류가 생기므로, 사용자를 인증할 수 있는 효과를 가진다. 이는 Github account info 또한 마찬가지이나, 사용자가 개인키를 안전하게 가지고 있을 경우 해킹 등의 공격에 보다 안전하다는 장점을 가진다.
