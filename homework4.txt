1. What is .gitignore? You can write the answer either in Korean or English.
- gitignore는 Git에서 특정한 파일이나 디렉토리 등 관리에서 제외할 파일들을 지정하는 파일을 말한다. 프로젝트를 진행하면서 자동적으로 생기는 로그파일 등의 파일을 git이 모두 추적하지 않도록 git의 관리에서 제외하는 것을 의미한다. 
2. Why do Github users need an SSH key pair? (Users can use either SSH key pair or github account info.) Please write down a brief explanation of SSH key.
 - github와 나의 작업공간(컴퓨터, 노트북)을 연동, 인증하는 작업이라고 생각할 수 있다. github repo에 나의 작업공간에서 일어나는 변화를 수시로 push 할 수 있게 하기 위해서는 github에 내 작업공간을 인증하는 것이 필요하고, 이 과정에서 사용되는 것이 SSH-key이다.
