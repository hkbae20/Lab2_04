1. What is .gitignore? You can write the answer either in Korean or English.
.gitignore란 git에서 특정 파일 혹은 디렉토리를 관리 대사에서 제회할 때 사용하는 파일이다.
즉, .gitignore 파일 안에 기입된 내용들은 git에서 관리하지 않겠다는 뜻이다.

2. Why do Github users need an SSH key pair? (Users can use either SSH key pair or github account info.) Please write down a brief explanation of SSH key.
아무나 저의 repository에 변경사항을 push하는 것을 방지하기 위해 git hub에 인증을 해야한다.
그 인증 작업에 사용되는 것이 SSH key이다. 오로지 인증된 유저들만이 저의 repository에 push할 수 있기 때문에
SSH key pair는 필요한 요소이다.
