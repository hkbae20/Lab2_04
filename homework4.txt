2. Why do Github users need an SSH key pair? (Users can use either SSH key pair or github account info.) Please write down a brief explanation of SSH key.
In order to use Github, you need to identify yourself by inputting a username and password. In order to 
avoid doing this everytime you use Github, an SSH key is a way to identify yourself without needing
a username and password.
