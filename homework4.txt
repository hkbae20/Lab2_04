1. What is .gitignore? You can write the answer either in Korean or English.
Git의 관리 대상에서 제외할 파일의 목록을 지정할 때 사용하는 파일.
.gitignore에 추가한 확장자나 폴더는 Git에 의해 무시된다. 

2. Why do Github users need an SSH key pair? (Users can use either SSH key pair or github account info.) Please write down a brief explanation of SSH key.
SSH key를 이용하면 유저네임과 비밀번호를 매 번 입력하지 않고 Github에 접속할 수 있게 된다. 그러면서도 높은 수준의 보안을 보장한다. SSH Key를 로컬 기기에서 생성하고 이를 등록하면, SSH 접속을 시도할 때 SSH 클라이언트가 이를 SSH 서버의 키와 비교하여 일치하면 접속을 허가하는 방식이다.
