2. Why do Github users need an SSH key pair? (Users can use either SSH key pair or github account info.) Please write down a brief explanation of SSH key.

비밀번호보다 높은 수주의 보안을 필요로 할 때 혹은 로그인 없이 자동으로 서버에 접속할 때 필요하다. SSH는 public key와 private key로 이루어지는데, private key는 local에 위치하고 public key는 remote에 위치한다. public key로 메시지를 암호화하며 암호화된 메시지는 그 쌍이 되는 private key로만 해석할 수 있다.
