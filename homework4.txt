2. Why do Github users need an SSH key pair? (Users can use either SSH key pair or github account info.) Please write down a brief explanation of SSH key.

SSH키는 사용자를 식별하는 키로, 매번 유저네임과 비밀번호를 입력할 필요 없게 해준다. public key는 깃허브와 같이 공유되는 서비스에 사용되고, private key는 개인 컴퓨터에 저장되어, 이 두 키가 일치하면 권한이 부여된다. 
결론적으로 본인인증(?)식별을 위해 key가 필요하다.
